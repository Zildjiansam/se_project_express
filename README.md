# WTWR (What to Wear?): Back End

This is the beginning of the backend for the WTWR project. It will implement, through the building of routes, controllers, and models, the language for getting lists of users and clothing items, creating/deleting new users, posting/deleting new clothing items, and liking/unliking specific clothing items within the app.

Technologies used:
-Node.js
-Express.js
-Mongo.db
-NPM/NPX
-ESLint

## Running the Project

`npm run start` — to launch the server

`npm run dev` — to launch the server with the hot reload feature

### Testing
